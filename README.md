# D-Project — Public Narrative Repository (v1.3)

This repository provides the **public-facing, non-technical description** of the D-Project.

It is designed for:
- citizens
- journalists
- researchers
- institutions exploring the concept at a high level

This repo does **not** include:
- proprietary governance rules
- internal metrics (NCC, SIL, ICF)
- implementation formulas
- institutional hosting documents

Those elements are maintained in a separate governance repository.

## Structure

- `manifesto/` — core public narrative (EN/FR)
- `architecture/` — high-level architecture, surplus ontology, coherence framework
- `coherence-dividend/` — full explanation of the Coherence Dividend and why it matters
- `notes-from-the-lab/` — essays and explanatory notes
- `licensing/` — licensing information for this repository
- `public-media/` — press-ready short descriptions (EN/FR)

All narrative content is provided under **CC BY-NC 4.0**, unless otherwise specified.
